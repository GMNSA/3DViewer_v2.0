SmartCalc v2.0

About program

Develop a program to visualise the 3D wireframe models.

  • C++ language of C++17 standard
  • Google style
  • The program developed according to the principles of object-oriented programming; the structured programming approach is not allowed

Description

    The graphical user interface contain:

  • button to select the model file and a field to output its name.
  • A visualisation area for the wireframe model.
  • Button/buttons and input fields for translating the model.
  • Button/buttons and input fields for rotating the model.
  • Button/buttons and input fields for scaling the model.
  • Information about the uploaded model - file name, number of vertices and edges.

Functional

  • 3D visual
  • Screenshots BMP, JPEG
  • GIF
